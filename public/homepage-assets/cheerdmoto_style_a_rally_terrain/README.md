# CHEERDMOTO Style A — Rally Terrain

This package is designed for direct use by Codex or a front-end developer.

## Start here
- `index.html`: immediately runnable stacked homepage preview
- `styles.css`: responsive image stack
- `preview/homepage_full.webp`: complete visual reference
- `slices/`: numbered, implementation-ready section images
- `assets/products/`: transparent PNG cutouts of the current CHEERDMOTO products
- `manifest.json`: filenames, dimensions and product mapping

## Filename convention
`NN_section-purpose.webp` — the numeric prefix preserves homepage order.

## Implementation recommendation
1. Reproduce each numbered slice as a semantic section.
2. Use the transparent product PNGs as foreground assets.
3. Keep headings, prices and specifications as live HTML text so they can be updated.
4. Use the full slice images as visual acceptance references, not as the only responsive implementation.

## Catalog represented
XTREME, XCEED, XCITE, XPLORE, XPLUS and SMART B02. Product details reflect the live site reviewed on 2026-07-06; confirm price and inventory at deployment.
